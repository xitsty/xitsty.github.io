Python Poker game

Description of it

A Poker game made in python one of my first project and only one made in python. It is basic working, uses a hashing algorithm that uses datetime to make a shuffle of the deck, uses down to millisecond to ensure that no duplicate deck can be made. It then stores the hashing seed used in a json file to it can be accessed later, to ensure players what happen and no cheating was involved. Plan to make it over into C# and implement it with the Bot. It only so fair deals the cards to the wanted player number and then tells who win.
