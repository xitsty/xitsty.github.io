C# Chatbot

Overview

This is a chatbot I built in C#. Right now, it’s fairly simple but already has a working login system to manage user access. The bot can also remember basic user details like name and age, which are added during the conversation.

Python Poker Game

Overview

This was one of my very first projects in Python. It’s a basic poker game that uses a datetime-based hashing algorithm to shuffle the deck down to the millisecond, making sure no two decks are ever the same. The seed is saved in a JSON file so results can be checked later, which adds a bit of fairness and transparency. Right now, the game deals cards to the chosen number of players and figures out the winner. I’d like to rebuild it in C# in the future and tie it into the chatbot as well.
